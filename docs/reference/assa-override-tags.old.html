ASSA - Override tags


Override tags

Advanced Sub Station Alpha override tags are used to add to or modify styles from the style header.
Override tags must appear within override blocks, which begin with { and end with }. Any unrecognized text within override blocks is silently ignored, so they are also commonly used for inline comments. Mixing comments and override tags in the same override block is not recommended.

Primary color

Set the primary color with two hex numbers for each of the three colors: blue, green, and red.
Note that it is not red/green/blue which is often used, but blue/green/red.

\c&H<bbggrr>& Set primary color
ExampleThis is {\c&H0000FF&}red This is red


 

Outline color

Set the outline color with two hex numbers for each of the three colors: blue, green, and red.
Note that it is not red/green/blue which is often used, but blue/green/red.
Setting the outline color has no effect if text does not have an outline (either set with \bord or style).

\3c&H<bbggrr>& Set outline color
ExampleThis has {\3c&H0000FF&}a red outline


 

Shadow color

Set the shadow color with two hex numbers for each of the three colors: blue, green, and red.

Setting the shadow color has no effect if text does not have an shadow (either set with \shad or style).

\4c&H<bbggrr>& Set shadow color
ExampleThis has {\4c&H0000FF&}a red shadow

Text shearing

Perform a shearing (perspective distortion) transformation of the text. A factor of 0 (zero) means no distortion.
Usually factor will be a small number; values outside the range -2 to 2 are unlikely to have desireable results.
Shearing is performed after rotation, on the rotated coordinates. The coordinate system used for shearing is not affected by the rotation origin.


\fax<factor>
\fay<factor>
Example{\fax1}fax1 text
Example{\fay1}fay1 text


 

Alpha

Set the alpha (transparency) between 0 and ff hex (0-255 decimal).
0 means opaque/fully visible, FF (255 decimal) is fully transparent/invisible (note: this is opposite of other programs!).


\alpha&H<value> Set text alpha
\1a&H<value> set primary alpha (text)
\3a&H<value> set outline alpha
\4a&H<value> set shadow alpha
Example w opaque box style{\alpha&HCC\1a0}Transparent box
Example{\an7\alpha&HCC}Watermark

Rotation origin

Set the origin point used for rotation. This affects all rotations of the line. The X and Y coordinates are given in integer script resolution pixels.
When there is no \org tag in a line, the rotation origin is implicitly the same as the position anchor point, e.g. center bottom of text for \an2.


\org(x,y) Fix the rotation origin at point (x,y)


 

Fade in/out

Fade-in and fade-out effect in milliseconds.
Adding a fade effect does not extend the duration of the line, but uses the start and end of the line's existing display time. For this reason, you should be careful that fadein + fadeout is not greater than the duration of the line. For example, if a line displayed for 4 seconds, the sum of fadein and fadeout should not be greater than 4000 ms.


\fad(fadein_ms,fadeout_ms) Display line with fadein and fadeout effect.
Example{\fad(500,500}Fade test
\fade(a1,a2,a3,t1,t2,t3,t4) Display line with complex fade-in and fade-out effect.
five-part fade using three alpha values a1, a2 and a3 and four times t1, t2, t3 and t4. The alpha values are given in decimal and are between 0 and 255, with 0 being fully visible and 255 being invisible. The time values are in milliseconds after the start of the line. All seven parameters are required.
  • Before t1, the line has alpha a1.
  • Between t1 and t2 the line fades from alpha a1 to alpha a2.
  • Between t2 and t3 the line has alpha a2 constantly.
  • Between t3 and t4 the line fades from alpha a2 to alpha a3.
  • After t4 the line has alpha a3.
ExampleFade {\fade(0,255,0,1000,1200,1300,1500)}(Advanced)


 

Move

Move text from one position to another.


\move(x1,y1,x2,y2) Move line
Example{\move(350,350,1500,350)}Move test
\move(x1,y1,x2,y2,t1,t2) Move line. "t1" is the millisecond in the line where the move starts, "t2" is the millisecond in the line where the move stops.
Example{\move(350,350,1500,350,500,2600)}Move test

These tags are supported: \fs \fsp \c \1c \2c \3c \4c \alpha \1a \2a \3a \4a \fscx \fscy \frx \fry \frz \fr \fax \fay \bord \xbord \ybord \shad \xshad \yshad \clip \iclip \be \blur

Example{\1c&HFFFFFF&\t(\1c&H0000FF&)}Color
