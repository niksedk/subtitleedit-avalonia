SubRip (*.srt)


SubRip is a very simple format created by the French programmer Brain in 2000, used for the program, also called, SubRip (written in Borland Delpi).
A very popular format today - and the default subtitle format in Subtitle Edit.
Can be embedded into Matroska files.

SubRip was originally just plain text, but today italic, bold, font, and even alignment tags are widely supported.
Time code format is "hours:minutes:seconds,milliseconds".

PROS: Very simple format, can be edited in any text editor. Very well supported in players/converters.
CONS: No official specification.
